# brand

The missing CLI for Firebase apps.

## Installation

### Service Account

`GOOGLE_APPLICATION_CREDENTIAL`

### Locale

You can set locale with the `DEFAULT_LOCALE` environment variable. Supported locales:

- `en` (Default)
- `sv`

### Usage

```sh
brand --help
```
